# 项目解析

## 项目组成
- web页面 （covid19-map）
- web端地图引擎 （covid19-map-view）
- 服务器 （covid19-server）

### web页面
- 采用 node.js 开发环境
- 使用 vue.js 核心库
- 使用 vue-router 做页面跳转
- 使用 ant-design-vue ui框架
- 使用 axios 做网络请求

### web端地图引擎
- 采用 node.js 开发环境
- 使用 vue.js 核心库
- 使用 vue-router 做页面跳转
- 使用 axios 做网络请求
- 使用 antV L7 地图引擎

### 服务器
#### 采用技术
- 采用 node.js 开发环境
- 使用 express 服务器框架
- 阿里云全国疫情数据接口
- 腾讯云全球疫情数据接口
- 使用 axios 向云服务发起请求
